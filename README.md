Golearn Packages [DB - Gorm]
====================================
Learn any packages to support web development.


### How to run
```shell
export DB_DSN=sqlite3://db.sqlite

# -- MIGRATION --
# make makemigrations name=create_users_table
make migrate

# -- RUN APPLICATIONS --
make runapi  # api application
make runweb  # web application
```
