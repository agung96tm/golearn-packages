Golearn Packages [Queue]
====================================
Learn any packages to support web development.


### How to run

```shell
make dependencies
make init

# run api application, worker, and scheduler
make runapi  
make workerapi
make schedulerapi

# run web application, worker, and scheduler
make runweb
make workerweb
make schedulerweb
```

Track your task and scheduler:
```shell
asynq dash
asynq stats
asynq cron ls  # scheduler
```
